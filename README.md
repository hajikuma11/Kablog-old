# Kablog

みんなも自分のカブ価を記録しよう！

# DEMO

まだないです。

# Features

あつまれどうぶつの森のカブ価を、それぞれ個人で記録するためのLINEBOTです。

# Requirement

"Kablog"を動かすのに必要なライブラリなどを列挙する

* huga 3.5.2
* Kabloghuga 1.0.2
